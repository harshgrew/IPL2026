📋 IPL Rules

1. Each team plays 14 matches in league stage.

2. Win = 2 points, Loss = 0 points.

3. No Result / Tie = 1 point each.

4. Teams ranked by total points.

5. If points equal → more wins rank higher.

6. If still equal → higher Net Run Rate (NRR).

7. Top 4 teams qualify for playoffs.

8. NRR = (Runs scored / Overs faced) - (Runs conceded / Overs bowled).
